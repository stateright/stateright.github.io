Stateright is a model checker for code written in the Rust programming language. The name refers to its goal of verifying that a system's overall state is always within specified correctness conditions.

This book explains how to use Stateright to verify the correctness of distributed system implementations, wherein nondeterminism manifests from events such as process crashes, network partitions, and clock drift. Model checking techniques have helped web service providers such as Amazon and Microsoft verify that distributed system designs are resilient to this nondeterminism, and Stateright goes further by enabling runnable implementations to be verified.
